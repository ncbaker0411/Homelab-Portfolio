# SIEM Deployment (Wazuh)

## Overview
This project implements a Security Information and Event Management (SIEM) system to monitor and analyze logs across the homelab environment.

## Real-World Scenario
Organizations rely on SIEM platforms to detect suspicious activity, monitor system health, and respond to security incidents.

## Objectives
- Centralize log collection
- Detect unauthorized access attempts
- Monitor system activity

## Use Cases
- Failed SSH login detection
- Log aggregation from multiple systems
- Alert visualization

## Skills Demonstrated
- SIEM deployment
- Log analysis
- Security monitoring
