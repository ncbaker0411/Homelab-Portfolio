# Network Design & IP Addressing

## Overview
This project focuses on designing and implementing a structured network architecture for a small-scale lab environment, simulating best practices used in enterprise networks.

## Real-World Scenario
In many small organizations, networks grow organically without structure, leading to IP conflicts, poor visibility, and difficult troubleshooting. This project addresses those issues by implementing a planned IP addressing scheme and logical segmentation strategy.

## Objectives
- Create a scalable IP addressing scheme
- Separate infrastructure, services, and clients
- Prepare for future VLAN segmentation

## Skills Demonstrated
- Network planning and design
- IP addressing strategy
- Infrastructure organization
